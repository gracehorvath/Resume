MasterCard

Java Developer Intern Summer 2014, St. Louis

I interened on the MasterPass team in the Emerging Payments department as a java developer intern. MasterPass is a digital wallet that allows consumers to store credit card and shipping information to create a seamless shopping experience. At the time, MasterPass was migrating from a single authorization web serivce to two new services, one for the business and the other for the consumer. I spent most of the summer creating mock classes that simulated the consumer authorization web serivce to allow developers to use this service on their local machines. I then moved on to creating an adapter class that would map the calls made from the merchant and the consumer to the appropriate authorization web service.
